# components/

**Path:** `apps/landing/src/components/`

Section components composing the marketing landing page. Each top-level file maps to one section of `App` (Navbar, Hero, Etymology, Features, HowItWorks, OpenSource, FinalCta, Footer) plus shared building blocks (`InstallCommand`, `LanguageSwitcher`, `NakirosLogo`).

## Subfolders

- [ui/](./ui/README.md) — UI primitives specific to the landing page.

## Files

- [Etymology.tsx](./Etymology.md) — "Etymology" section explaining the origin of the name "Nakiros".
- [Features.tsx](./Features.md) — "Features" section rendering a 2-column grid of feature cards.
- [FinalCta.tsx](./FinalCta.md) — Final call-to-action section with the install command and GitHub link.
- [Footer.tsx](./Footer.md) — Page footer with logo, tagline, and copyright.
- [Hero.tsx](./Hero.md) — Top-of-page hero block with eyebrow badge, headline, and primary install command.
- [HowItWorks.tsx](./HowItWorks.md) — "How it works" 3-column numbered timeline.
- [InstallCommand.tsx](./InstallCommand.md) — Copy-to-clipboard install command pill with version badge.
- [LanguageSwitcher.tsx](./LanguageSwitcher.md) — Compact EN/FR locale toggle rendered in the Navbar.
- [NakirosLogo.tsx](./NakirosLogo.md) — Inline SVG of the Nakiros constellation logo.
- [Navbar.tsx](./Navbar.md) — Sticky top navigation bar with logo, anchor links, locale toggle, and install CTA.
- [OpenSource.tsx](./OpenSource.md) — "Open source" two-column section with values bullet list and GitHub link.
