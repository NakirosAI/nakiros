# lib/

**Path:** `apps/landing/src/lib/`

Shared helpers for the landing page: classname combinator and the npm version-fetch hook used by `InstallCommand`.

## Files

- [useNpmVersion.ts](./useNpmVersion.md) — React hook fetching the latest published version of an npm package with dist-tag fallback.
- [utils.ts](./utils.md) — Misc helpers — currently the `cn` Tailwind classname combinator.
