# packages/

**Path:** `packages/`

Internal packages consumed by the Nakiros apps. These are `@nakiros/*`-scoped workspace packages; they are not published to npm individually — only `@nakirosai/nakiros` (the app in `apps/nakiros`) is.

## Subfolders

- [shared/](./shared/README.md) — `@nakiros/shared` — shared TypeScript types and the canonical IPC channel registry.
