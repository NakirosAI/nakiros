UX — Events Ledger & Traçabilité

Generated by Nakiros on 2026-03-27.

Layout

Header + TabView (ACK Events / Failed Transactions) + Sidebar drawer de détail d'une transaction.

Composants principaux

  • TabView : onglets ACK Events / Failed Transactions
  • Sidebar PrimeVue : drawer de détail latéral (LedgerTransactionDrawer)
  • ConfirmDialog : confirmation avant replay réel

Interactions

  • Clic sur une transaction → ouverture du drawer
  • Bouton Dry-run → prévisualisation de l'événement
  • Bouton Exécuter → ConfirmDialog + replay réel

Incohérences UX identifiées
